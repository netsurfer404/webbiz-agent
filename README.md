# WebBiz Agent 🚀

An AI-powered autonomous business agent that runs a complete website building 
business — finding potential clients, reaching out to them, and building 
their websites — all from a single interface.

Built on top of Claude AI (Anthropic), Resend, and Supabase.

---

## What It Does

- 🔍 **Prospect** — Finds businesses around the world that don't have a website
- 📊 **Analyze** — Researches each business and builds a value pitch
- 🏗 **Build** — Generates a full professional website for the client using AI
- ✉️ **Outreach** — Writes and sends personalised cold emails via Resend
- 📞 **Call Scripts** — Generates phone scripts for voice outreach
- 📋 **Pipeline** — Tracks every lead and its status in real time via Supabase

---

## Tech Stack

| Service | Purpose |
|---|---|
| Claude AI (Anthropic) | Lead generation, email writing, site building |
| Resend | Real email delivery from buildmyweb.shop |
| Supabase | Lead database and pipeline storage |
| GitHub Pages | Hosts the agent interface |
| GitHub Actions | Runs automated daily prospecting (coming soon) |

---

## Setup

1. Clone or download this repo
2. Open `index.html` in your browser
3. Go to the ⚙️ Settings tab
4. Enter your API keys:
   - Resend API Key
   - Supabase URL
   - Supabase Anon Key
5. Click Save Config and Test Connections
